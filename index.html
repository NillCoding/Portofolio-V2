Game Snake

Cara Bermain:Gunakan tombol panah untuk menggerakkan ular. Makan makanan (kotak merah) untuk menambah skor dan panjang ular. Hindari menabrak dinding atau badan ular sendiri. Kontrol menggunakan tombol panah pada keyboard. Skor bertambah setiap kali ular memakan makanan. Game berakhir jika ular menabrak dinding atau tubuhnya sendiri. Wajib menggunakan laptop/pc. Link code: https://replit.com/@nillmoderog/game-snake-1#index.html, Code ini bisa di copy. code ini menggunakan code HTML,CSS dan Java Script
